# SSH Hardening CybersecurityLab
I have run the virtual machine Oracle Virual Box and I have configured my network home lab. Throughout this experience I have learnt to harden a netowrk and block doors to intruders.

What I did in this lab:
Changed default SSH port 22 to port 2222
Edited the SSH configuration file /etc/ssh/sshd_config
Enabled UFW firewall
Verified firewall rules using UFW status verbose
Configured the firewall to allow traffic only on the custom SSH port 2222
Running status checks on ports in the network
